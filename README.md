# greenway-recruit

## Vite + React + TypeScript

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

The poster component lives in:

- `src/GreenwayPoster.tsx`
- `src/GreenwayPoster.css`

The app entry is:

- `src/App.tsx`

Example usage:

```jsx
import GreenwayPoster from './GreenwayPoster';

export default function App() {
  return <GreenwayPoster />;
}
```
